Brazo Robótico 3D

Proyecto de diseño, planificación y desarrollo de un brazo robótico impreso en 3D, documentando todo el proceso desde la idea hasta el prototipo funcional.

Estado actual

El proyecto se encuentra en fase inicial. Se ha realizado un croquis conceptual y se está definiendo la arquitectura mecánica, número de ejes y tipo de actuadores necesarios.

En qué trabajo +
Frontend avanzado, CSS creativo, APIs, diseño técnico y sistemas web.
Filosofía +
Menos ruido. Más intención. Cada detalle cuenta.

Objetivos

Diseñar un brazo robótico modular, escalable y documentado, aplicando conocimientos de mecánica, electrónica y control.

Materiales

Se prevé el uso de PLA o PETG para las piezas estructurales, junto con tornillería estándar, ejes metálicos, motores y un microcontrolador para el control del sistema.

Próximos pasos

Modelado 3D detallado, simulaciones de movimiento, impresión de prototipos y primeras pruebas funcionales.
